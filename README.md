## Requirements

NPM & Gulp

## Installation

`$ git clone https://github.com/AlexandraKlein/gulp-start.git`

`$ cd gm`

`$ sudo npm install`

`$ sudo npm install --global gulp-cli`

`$ sudo npm install gulp-cssnano --save`

## Start Local Environment

`$ gulp`


## Building Dist Folder

`$ gulp build && gulp useref`
